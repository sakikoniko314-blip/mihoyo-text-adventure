# 这是一个原神崩铁的文字冒险游戏

整个游戏基于由游戏内文本构建的完整文本库。这使得 AI 能够完整地获取到相当全面的世界观和角色设定。选择世界和起始地点，AI 续写故事，你选选项推进。

---

## 为什么

让 AI 跑团最大的问题是出戏——"你好旅行者，我叫温迪，是璃月的天权星"。因为这个项目不是裸调 AI，而是把原神和崩铁游戏里的角色故事、语音、书籍、物品描述、任务对话爬下来，建了 FTS5 全文索引。AI 写故事时提到哪个角色，系统自动检索相关设定喂进去。温迪不会再说自己来自稻妻了。

## 选项不是敷衍的

最早版本 AI 只会给"继续前进""观察四周""转身离开""试着呼喊"。改了 n 版之后拆成双 AI 架构：一个专门写故事，一个专门设计选项。现在选项长这样（璃月港码头）：

- 快步走向那艘卸货的商船，假装帮忙搭把手，趁机打听那些奇怪的箱子
- 沿着礁石上的脚印摸过去，趁涨潮前看看石缝里藏着什么
- 凑近那个争吵的商人，装作不经意插嘴套话
- 扑向快被海风吹走的纸条，捡起来发现背面有暗红色痕迹

每个选项有具体动作、有悬念、有后果暗示。

## 冒险有结局

早期没结局，故事永远在"你要继续吗"死循环。后来加了个机制：AI 判断当前剧情到了自然高潮，输出 `【冒险结束】` 标记，触发独立终章叙事，给冒险一个收束。
